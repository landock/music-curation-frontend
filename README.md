## Prerequisites
```
$ npm i -g @storybook/cli
```

## LOCAL DEV

### [TAB 1]
```
$ nvm use 8.9.4
```
### [TAB 2]
React App runs on http://localhost:3000/
```
$ nvm use 8.9.4 && yarn start
```

### [TAB 3]
Storybook runs on http://localhost:9009/
```
$ nvm use 8.9.4 && yarn run storybook
```

### Testing
```
$ yarn test
```
